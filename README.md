# simple-application


```
# clone repo
git clone https://github.com/mrsecret911/simple-application.git

# change directory 
cd simple-application

# install node modules
npm install

# start front-end server
npm start


# start beck-end server

# install nodemon
npm install -g nodemon

# run beck-end server from simple-application/server
nodemon index.js
```
